## 交集选择器

```html
作用：选中同时复合多个条件的元素
语法：选择器1选择器2选择器3选择器n{}
注意点：
    交集选择器中如果有元素选择器，必须使用元素选择器开头
<style>
  div.red{font-size: 30px;}
	.a.b.c{color: blue}
</style>
```

## 选择器分组（并集选择器）

```html
作用：同时选择多个选择器对应的元素
语法：选择器1,选择器2,选择器3,选择器n{}
#b1,.p1,h1,span,div.red{}
<style>
  h1, span{color: green
}
</style>
```
